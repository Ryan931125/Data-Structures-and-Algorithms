# Data Structures and Algorithms Projects

This repository contains my DSA course programming projects in C. Each folder corresponds to one problem and is named by the main algorithm or data structure used.

## Project Folders

- `Stack_ShuntingYard`: Infix to postfix conversion and expression evaluation with stacks.
- `BinarySearch_Trace`: Binary search with per-step range tracing.
- `KMP_PrefixSimilarity`: Prefix-based similarity scoring using KMP and prefix function.
- `LinkedList_DeckOperations`: Multi-deck card simulation with linked-list operations.
- `UnionFind_PathCompression`: Disjoint sets with path compression and union by rank.
- `RankSimulation_BinarySearch`: Rank/power simulation with binary-search-based queries.
- `DynamicGraph_DungeonTraversal`: Dynamic dungeon graph operations with planning queries.
- `Kosaraju_SCC_DAGPath`: SCC decomposition and condensed DAG path feasibility.
- `FloydCycleDetection_LinkedList`: Cycle detection on linked structures.
- `DFS_EdgeClassification`: DFS edge classification in directed graphs.
- `InsertionSort_MoveBudget`: Insertion sort under move budget.
- `MaxHeap_RepeatedInsert`: Max-heap construction by repeated insertion.
- `BinomialHeap_PrintQueue`: Printer queue management with binomial heaps.
- `QuickSort_Budget`: Quick sort under swap budget.
- `RedBlackTree_Rotations`: Red-black tree rotations and node queries.
- `SelectionSort_Budget`: Selection sort under swap budget.
- `RabinKarp_KMP_Multiline`: Multi-line string matching with hashing + KMP.
- `Backtracking_ZigZagPermutations`: Lexicographic zig-zag permutation generation with pruning.
